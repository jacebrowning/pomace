# Setup

## Requirements

- Python 3.8+

## Installation

Install it directly into an activated virtual environment:

```text
$ pip install pomace
```

or add it to your [Poetry](https://poetry.eustace.io/) project:

```text
$ poetry add pomace
```

# Usage

After installation, the package can imported:

```text
$ python
>>> import pomace
>>> pomace.__version__
```
